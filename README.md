RPN计算器使用说明<br>
### 一.安装 
确保你已经安装了 g++ 编译器。如果你使用的是 Linux 系统，可以通过以下命令进行安装：
sudo apt update<br>
sudo apt install g++<br>
### 二.编译
可以使用以下命令进行编译：
g++ -o rpn_calc rpn_calc.cpp<br>
### 三.运行
编译成功后，运行以下命令启动程序：
./rpn_calc<br>
### 四. 基本使用示例
启动程序后，输入一个逆波兰表达式（例如 5 5 +）并按回车键，程序会计算并显示结果。<br>
例：> 3 4 + 2 * 7 /<br>
结果: 7<br>
该示例计算 ( 3 + 4 ) * 2 / 7 ，并输出结果 7<br>
输入 q 退出程序。<br>
### 五. 所有支持的操作说明<br>
该程序支持以下基本操作：<br>
加法 +：计算两个操作数的和。<br>
减法 -：计算两个操作数的差。<br>
乘法 *：计算两个操作数的积。<br>
除法 /：计算两个操作数的商，除数不能为零。<br>
平方根 sqrt：计算栈顶元素的平方根，要求输入的数为非负数。<br>
操作数包括整数和浮点数<br>
### 六. 错误代码和异常说明<br>
1. 栈为空错误<br>
当栈为空时按回车键，程序会抛出异常并显示以下错误信息：<br>
错误: 栈为空。<br>
2. 除零错误<br>
当执行除法操作时，如果除数为零，程序会抛出除零异常：<br>
错误: 除数不能为零。<br>
3. 无效输入错误<br>
当输入的内容无法解析为数字或有效操作符时，会抛出异常并显示以下错误信息：<br>
错误: 无效输入: x<br>
4. 平方根错误<br>
当尝试对负数进行平方根计算时，会抛出异常并显示：<br>
错误: 不能对负数求平方根。<br>
### 七、输入输出

计算过程：（3.8 + 4）* 2.9 / 7 = 7
